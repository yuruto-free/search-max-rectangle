# Search Max Rectangle

## C language version
### Environment
* compiler

    GCC

* library

    math.h

* required command

    make

### Build program
Run the following command.

```sh
make
```

### Execute program
Execute the following command.

```sh
./search-max-rectangle
```

### Remove execution program
Run the following command.

```sh
make clean
```

## Python version
### Environment
* version

    python 3.7.3

* library

    math

### Execute program
Run the following command.

```sh
cd python
python3 search_max_rectangle.py
```
